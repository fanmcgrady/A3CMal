# 恶意软件家族分类对抗

## 使用方式
Python推荐使用3.7
```
git clone git@github.com:fanmcgrady/A3CMal.git
cd A3CMal
pip install -r requirements.txt
```

拷贝pe数据集到Dataset/pe/test_malconv、Dataset/pe/train_malconv
